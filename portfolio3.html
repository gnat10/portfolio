5 Kata || a Wordle Clone

An Indonesian version of Wordle where you need to guess a 5 letter Indonesian word instead. Challenged myself to make a guessing game in JavaScript. The hardest part was to make sure each tile have the correct colour after a guess as well as to limit each word to be 5 letter before moving to the next row. Some of it still need to be improved like how the keytiles doesn't change at the same time after a guess. However the main function is done.
